# Enrichers

Questa cartella conterr? gli arricchitori dei dati progetto.

Obiettivo:
- partire da CUP / CIG / committente / comune;
- cercare fonti pubbliche aggiuntive;
- recuperare, dove disponibile:
  - azienda incaricata / aggiudicatario;
  - P.IVA / codice fiscale;
  - indirizzo/sede;
  - PEC/email/telefono pubblici;
  - RUP o referente amministrativo;
  - importo e data aggiudicazione;
  - URL fonte dell'esito o determina.

Fonti previste:
- ANAC / BDNCP;
- Servizio Contratti Pubblici;
- albi pretori;
- siti istituzionali degli enti;
- atti di gara / determine / esiti.
